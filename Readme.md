### How to clone

```git clone --recursive https://github.com/TheSpyGeek/ImGUITemplate.git```

### Dependices

To install dependices :

```sudo apt-get install -y build-essential cmake xorg-dev libgl1-mesa-dev libfreetype6-dev```
